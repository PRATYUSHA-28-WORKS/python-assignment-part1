
# Student Grade Tracker - Python Basics

This repository contains **Part 1** of the Python Assignment, focusing on data cleaning, control flow, and string manipulation.

## Project Overview
The Student Grade Tracker is a command-line application that processes raw student data, analyzes marks, and generates a class performance report.

### Key Features
* **Data Cleaning:** Transforms messy strings and inconsistent casing into structured, normalized data (Title Case, integer conversion, list parsing).
* **Grade Analysis:** Uses loops and conditional logic to assign grades (A+, A, B, C, F) and calculate statistics like averages and highest/lowest scores.
* **Interactive Input:** Features a `while` loop system to dynamically add new subjects with built-in input validation.
* **Class Reporting:** Generates a formatted summary table showing pass/fail status and class-wide performance metrics.
* **String Utilities:** Demonstrates advanced string methods like stripping, replacing, and sentence-level manipulation.
